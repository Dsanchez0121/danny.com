## How to write a post

1. Create a new file in `content/posts/` — filename becomes the URL slug
   - Example: `content/posts/shanghai-notes.mdx` → `/blog/shanghai-notes`

2. Add frontmatter at the top (the `---` block). Required fields:

```yaml
---
title: "Your title"
date: "2026-07-07" # YYYY-MM-DD
excerpt: "One sentence that shows on cards"
coverImage: "/images/your-cover.jpg" # put image in public/images/
location: "Optional, like Shanghai"
tags: ["lore", "languages"]
published: true # false = draft, hidden
---
```

3. Write in markdown below. You can use **bold**, *italic*, lists, quotes, code, etc.
